## 📖 À propos

**Dwaya** (دواء — *médicament* en arabe, slogan *Matnsach Dwak* = *N'oublie pas ton médicament*) est un prototype d'application mobile destinée aux seniors marocains sous traitement chronique. L'app combine reconnaissance par photo, rappels multimodaux (audio en Darija, visuel et haptique), suivi de l'observance et partage familial pour réduire les oublis de prise.

Ce dépôt contient le **prototype interactif** issu d'une démarche Lean Startup (*Build → Measure → Learn*).

## 🧪 Démarche Test & Learn

Le prototype a été testé auprès de **6 séniors âgés de 60 à 75 ans**, sous traitement chronique, lors de sessions individuelles de 45 minutes. Quatre hypothèses critiques ont été évaluées :

| # | Hypothèse | Statut |
|---|-----------|--------|
| **H1** | Les seniors comprennent l'interface dès le 1er usage sans accompagnement | 🟡 Partiellement validée |
| **H2** | Les seniors comprennent qu'il faut photographier la boîte du médicament | ✅ Validée |
| **H3** | Les rappels audio + visuels réduisent les oublis de prise | ✅ Validée |
| **H4** | Le partage de données avec un proche est jugé acceptable et rassurant | ❌ Invalidée — Pivot |

Les apprentissages issus de cette phase ont guidé l'itération vers la version améliorée.
